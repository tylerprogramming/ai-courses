# PRODUCT FAQ

### 1. How long does installation take?
A typical installation takes 4–6 hours, depending on roof type and panel layout.

### 2. Does the system include a battery?
A battery pack is optional but recommended for backup and peak-hour storage.

### 3. Can I monitor energy output?
Yes — the accompanying mobile app includes real-time dashboards,
notifications, and historical performance graphs.

### 4. What is the warranty?
All solar panels come with a 25-year performance guarantee.
